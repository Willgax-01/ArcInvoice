# ArcInvoice

ArcInvoice is a modern invoicing dApp prototype built for the Arc ecosystem.

## Features

- Wallet Connection
- Invoice Creation
- QR Code Payments
- Dashboard Analytics
- Transaction History
- Mock Arc Testnet Integration

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Ethers.js

## Vision

ArcInvoice aims to help freelancers and small businesses create invoices and receive stablecoin payments on Arc Network.
